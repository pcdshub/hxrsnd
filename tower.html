Base Tower Class¶

All the towers inherit fromt the TowerBase class.

class hxrsnd.tower.TowerBase(prefix, name=None, pos_inserted=None, pos_removed=None, *args, **kwargs)¶

Base tower class.

check_status(energy=None, length=None, no_raise=False)¶

Checks to make sure that all the energy motors are not in a bad state. Will include the delay motor if the delay argument is True.

Parameters

  • energy (float or None, optional) – Energy to set the tower to.

  • length (float or None, optional) – Length to set the delay stage to. (Doesnt apply for cc towers)

  • no_raise (bool, optional) – Do not re-raise the attribute error for delay parameters.
